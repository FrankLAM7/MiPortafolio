Proyectos Móviles

Para poder probar algunos de mis proyectos es necesario seguir unos sencillos pasos.



Una vez descargada, ingresas y seleccionas la opción "Scan QR Code" , escaneas los siguientes codigos QR de mis aplicaciones.

Taxy Usuarios

Este proyecto imita la funcionalidad de aplicaciones de servicio de taxy como Uber, Indriver, etc. Fue desarrollado en REACT NATIVE y la base de datos en FIREBASE, es una aplicación hibrida la cual es una parte de 2 aplicaciones que interactuan entre sí.

Taxy Conductores

Esta es la segunda aplicación que interactua con la anterior, en esta se toma el papel de conductor el cual recibe alertas de los usuarios que solicitan una unidad de taxy, para ingresar es necesario poner como usuario: prueba@taxy.com contraseña: 123456

Simulación App

Este fue un proyecto en el cual tuve que hacer una simulación donde el celular enviaba alertas al usuario final, para ingresar puedes utilizar cualquier acceso no esta validada con alguna base de datos.
