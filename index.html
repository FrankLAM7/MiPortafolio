Aquí encontraran algunas aplicaciones móviles que he realizado, mediante el lenguaje de programación ReactNative puedo crear aplicaciones híbridas que se pueden renderizar tanto para Android como IOs.

Tienda Online

Esta aplicación web fue creada en ANGULAR, muestra la interfaz de una tienda que muestra diferentes categorias y productos, esta información de muestra se obtienen de una base de datos alojada en FIREBASE de Google.

Peliculas App

Aplicación creada en ANGULAR la cual muestra una sencilla interfaz de bienvenida, donde hay un boton ingresar, en el cual demuestro que cuento con los conocimientos necesarios para integrar el loggin con el api de google. Una vez logeado, muestra una interfaz con las peliculas mas populares a la fecha, esta información es obtenida de una API externa, tambien tienes un buscador de peliculas el cual busca en el API y devuelve los resultados a la busqueda. Cada pelicula tiene un boton rojo de información el cual te lleva a otra pantalla donde puedes encontrar más información de la misma.
